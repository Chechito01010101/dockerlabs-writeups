# 🐳 DockerLabs Writeups

Repositorio donde documento mis **writeups**, notas técnicas y scripts desarrollados durante la resolución de máquinas de **DockerLabs**. El objetivo de este proyecto es fortalecer mis habilidades en **Pentesting**, **Hacking Web**, **Explotación de Vulnerabilidades** y **Escalada de Privilegios**, siguiendo una metodología estructurada y documentando el proceso de aprendizaje.

> **⚠️ Disclaimer:** Este repositorio tiene fines exclusivamente educativos. Todo el contenido corresponde a laboratorios autorizados de DockerLabs y está destinado al aprendizaje de técnicas de ciberseguridad ofensiva.

---

## 🎯 Objetivos

- Documentar cada laboratorio de forma técnica.
- Reforzar conocimientos de Ethical Hacking.
- Mejorar habilidades de Pentesting Web y de Infraestructura.
- Practicar metodologías de reconocimiento, explotación y post-explotación.
- Crear un portafolio técnico para demostrar habilidades prácticas.

---

## 📌 Estado del proyecto

Este repositorio se encuentra en constante actualización conforme voy resolviendo nuevas máquinas de DockerLabs.

---

## ⚖️ Uso responsable

La información publicada tiene un propósito exclusivamente educativo. No debe utilizarse para realizar actividades no autorizadas sobre sistemas o redes de terceros.
